# 毎日少しずつ進めるための成果物ルール

## 基本形

毎日すべてを大きく進めようとしない。

- **主成果物 1つ**: 45〜90分。あとで見返せる形まで残す。
- **短く触れるもの 2つまで**: 各15〜20分。続きが分かる一行を残して終える。
- **身体の手入れ 1つ**: 散歩、運動、休養など。その日の体調に合わせる。

主成果物ができたら、その日は前進したとみなす。

## 成果物の条件

「作業した」ではなく、明日の自分や他の人が見て使えるもの。

- **AI**: 小さく動く試作品、使えるプロンプト、検証結果のメモ
- **POP**: 撮影リスト、連絡文案、30秒の語り動画、予算表、構想図
- **音楽**: ワンコーラスの録音、歌詞の選択、アレンジメモ
- **教育・教会**: 引き継ぎメモ、手順書、予定表、教材1点

## 体力が必要なことを先にする

次のようなものは、AIで置き換えず、早めに実行または素材化する。

1. 現地へ行く、人に会う、撮影する
2. 歌う、演奏する、自分の声や姿を記録する
3. 長時間の移動や身体を使う活動

AIは、これらを実行しやすくする準備、編集、整理に使う。

## 明日の試運転

- **主成果物**: 「8月フィリピンで撮るものリスト v0.1」を1枚作る
- **AIに触れる**: リストをAIに見せ、抜けを3点だけ出してもらう
- **音楽に触れる**: 既存曲を1曲だけ聴き、今後残したい形を一行書く
- **身体**: 20分歩く、または疲れていれば早く休む

## 1日の記録

```markdown
### YYYY-MM-DD
- 今日の主成果物:
- 短く触れたもの:
- 体調（5段階）:
- 明日の最初の一手:
```

## 週末の確認

時間ではなく、残った成果物を数える。数が少ない週は、気合いを増やさず、主成果物をさらに小さくする。
