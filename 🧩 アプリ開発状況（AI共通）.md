## 1. 開発中のアプリ一覧

### A. 「俺は世界を変える！」個人用実行アプリ

- 本体: [[俺は世界を変える！.html]]
- 仕様: [[俺は世界を変える！アプリ仕様]]
- 確認: [[俺は世界を変える！アプリ確認シート]]
- 状態: 試作 v0.1相当。画面・ローカル保存・タスク追加/完了/削除・進捗表示は実装済み。
- localStorage key: `change-the-world-v05`

目的:
- 大きな目標と今日できる小さな行動をつなぐ。
- タスク完了時に、POP三段階または活動基盤がどれだけ進んだかを返す。

主な構造:
- 三段階: カフェの設立 / 孤児院の設立 / いい人システムの確立
- 活動基盤: 学校 / 教会 / 資金 / 時間 / 体力
- POP第一期ロードマップ:
  1. 送金と共同経営確認
  2. 建設台帳
  3. 8月建設参加・撮影
  4. 通常営業の記録
  5. 3月一週間営業
  6. 共有・検証

実装済みの主な機能:
- 三段階と活動基盤の進捗表示
- POP第一期ロードマップ表示
- 今日の一歩表示
- タスクの追加、完了、削除
- タスク完了時のフィードバック表示
- 完了履歴
- ブラウザ内保存
- 学校事業の現在値と2年後見込みの簡易メーター

現在の課題:
- 「大目標を動かした」と感じる面白さがまだ弱い可能性がある。
- 初期タスクが今の送金前クリティカルと完全には同期していない可能性がある。
- 活動基盤が重く見えすぎる場合は、表示を軽くする必要がある。
- 実画面での確認がまだ十分ではない。

次に見ること:
- [[俺は世界を変える！アプリ確認シート]] に沿って、コードではなく画面と操作で確認する。
- POP第一期の初期タスクを、現在の実務に合わせて入れ替える。
- 送金前ゲート補助アプリで得た構造を、本体へ統合するか判断する。

### B. POP送金前ゲート作成補助アプリ

- 本体: [[POP送金前ゲート作成補助アプリ.html]]
- 関連判断メモ: [[POP送金前の優先順位と判断ゲート]]
- 状態: v0.0.5。送金前に必要な家庭ゲート・資金性質ゲートの下書きを作るための独立HTML試作。
- localStorage key: `pop_gate_builder_v0`

目的:
- 家庭ゲート v0.1 と資金性質ゲート v0.1 を作る。
- 最初から全項目にAI下書きを入れておき、ユーザーが違うところだけ直せるようにする。
- 作成した内容をObsidian用Markdownとしてコピーまたは保存できるようにする。

対象ゲート:
- 家庭ゲート
- 資金性質ゲート

実装済みの主な機能:
- ゲート選択
- 質問一覧
- 5分 / 前進 / 整理の答え方ガイド
- AI下書きプリセット
- 回答の自動保存
- Markdown下書き生成
- コピー
- Markdown保存
- 下書きに戻す

現在の課題:
- 実際に入力してみて、質問数が多すぎないか確認する必要がある。
- AI下書きが強すぎる箇所、事実と違う箇所、尚子さんへの説明として出しすぎな箇所を修正する必要がある。
- 6月末送金までの実務に対し、どこまでこのアプリを使うか判断が必要。

次に見ること:
- 家庭ゲートの「家庭ゲートを一言で定義する」
- 「この150万円で何が生まれるのか」
- 資金性質ゲートの「150万円は何に近いか」
- 税理士等に聞く質問

## 2. 現在の開発方針

いまは、アプリそのものを増やす時期ではなく、送金前クリティカルを進める時期。

優先順位:

1. デニス返信待ちと送金前確認
2. 尚子さん向け報告・説明責任
3. 150万円の会計・税務上の名目
4. 送金方法と受領記録
5. 必要に応じて、送金前ゲート補助アプリを使って下書きを作る
6. その後、「俺は世界を変える！」本体に統合するか判断

開発では、まず「動くものを増やす」より、「何を実行させたいか」を絞る。

## 3. アプリ同士の関係

```text
POP第一期ロードマップ
  ├─ 送金前ゲート

  │         ├─ 家庭ゲート v0.1
  │         └─ 資金性質ゲート v0.1
  │
  └─ 日々の実行
       └─ 「俺は世界を変える！」個人用実行アプリ
            ├─ 今日の一歩
            ├─ 三段階メーター
            ├─ 活動基盤
            └─ 完了履歴
```

送金前ゲート補助アプリは、本体アプリに入れる前の「実地テスト」。  
本体アプリは、POP第一期全体を日々のタスクへ落とすための土台。
