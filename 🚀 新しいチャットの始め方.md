# 🚀 新しいチャットの始め方

> 新しいチャットは**毎回まっさら**。Vaultフォルダを接続していれば `CLAUDE.md`（Codexは `AGENTS.md`）が読み込まれ、**`ynp97` という合図があるときだけ**フル文脈を読む。雑用ではトークンを使わない。

## 2つのモード

- **戦略モード**：最初に **`ynp97` だけ**を送る。→ AIが 🧭→📌→🗒→🔭 を読んで戦略モードに入り、その相談が終わるまで継続。続けてテーマを送る。終了時に相談ログへ書き戻す。
- **軽量モード**：合図なし＝普通の作業・質問。→ 文脈は読まず手早く対応。トークン節約。
- **記録したいとき**：軽量モードでも「**記録して**」と言えば、その要点を 🗒 相談ログ に残す。

## 手順（Cowork / Claude）

1. 新しいチャットで **Vault フォルダ（Obsidian Vault）を接続**する（既定の作業フォルダにしておけば毎回不要）。
2. 戦略を練るなら、最初に `ynp97` だけを送る。AIが読み込みを終えたらテーマを送る。雑用ならそのまま頼む。

## 起動フレーズ（保険・コピペ用）

戦略モードに入るときは、まずこれだけを送る：

```
ynp97
```

## Codex の場合

- この Vault フォルダで Codex を起動すれば、直下の `AGENTS.md` が自動で読み込まれる。
- 戦略相談では最初に `ynp97` だけを送り、読み込み完了後にテーマを送る。

## うまく通じないときのチェック

- フォルダを接続したか？（未接続だと CLAUDE.md / AGENTS.md が無く、文脈ゼロになる）
- 起動フレーズで4ファイルを読ませたか？
- 前回の続きは 🗒 相談ログ の末尾にあるか？（無ければ前回のAIが書き戻していない＝終了時に追記を促す）
