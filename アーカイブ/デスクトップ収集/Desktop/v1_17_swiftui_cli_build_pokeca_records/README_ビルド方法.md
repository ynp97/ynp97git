ポケカ戦績 SwiftUI版 v1.17

手順:
1. build.command を右クリック → 開く
2. ビルド完了後、PokecaRecords_v1.17.app が生成されます
3. 自動で起動テストします
4. 開けない場合は、ターミナルで以下を実行してください

cd "このフォルダのパス"
./PokecaRecords_v1.17.app/Contents/MacOS/PokecaRecordsLauncher

ログ:
~/Library/Application Support/PokecaRecordsSwiftUI/build.log
~/Library/Application Support/PokecaRecordsSwiftUI/app.log
~/Library/Application Support/PokecaRecordsSwiftUI/app_error.log

v1.17の変更:
- .app名を英数字にしてFinder互換性を上げました
- Finder起動時にランチャーを経由し、エラーをログに残します
- xattr -cr で隔離属性を外します
- PkgInfoを追加しました
- ビルド後に自動で open します


v1.17: 起動時にNSAlertを出してクラッシュする問題を修正。エラーは画面上部とapp_error.logに表示します。


更新 v1.17:
- 自分のデッキと相手デッキを別々に登録できるようにしました。
- 入力候補も自分用・相手用で分離しました。
- 既存データベースのデッキ表を自動移行します。
