2025・11・22（土）ヨシュア記7:16-26 Morning Dew

この段落は、前半（7:1--15）で神が宣言されたことが、具体的な歴史的行為として実行される部分です。

流れとしては

① 7:16--18　「くじ」によるアカンの特定

② 7:19--23　アカンの告白と、罪の実物が暴かれる過程

③ 7:24--26　アカンとその所有物への裁きと、「アコルの谷」という記念

として読むことができます。

**7:16--18　くじによる「絞り込み」と、罪が逃れられないことの強調**

**Josh. 7:16** 翌朝ヨシュアは早く起き、イスラエルを部族ごとに進み出させた。ユダ部族がくじで取り分けられた。

**Josh. 7:17** ユダの諸氏族を進み出させると、ゼラフ人の氏族がくじで取り分けられた。ゼラフ人の氏族を男一人ひとり進み出させると、ザブディがくじで取り分けられた。

**Josh. 7:18** ザブディの家族を男一人ひとり進み出させると、ユダ部族のゼラフの子ザブディの子カルミの子のアカンが、くじで取り分けられた。

●「翌朝早く起き」

ヨシュアはまたも「朝早く起き」て（3:1、6:12も同じ）、主の命令に即座に従います。

・罪の問題を先延ばしにせず、「翌日すぐ」扱う姿勢です。

**Josh. 7:14** 明日の朝、部族ごとに進み出よ。**主**がくじで取り分ける部族は氏族ごとに進み出、**主**がくじで取り分ける氏族は家族ごとに進み出、**主**がくじで取り分ける家族は男一人ひとり進み出よ。

**Josh. 7:15** 聖絶の物のことでくじで取り分けられた者は、彼も彼に属するすべてのものも、火で焼かれなければならない。彼が**主**の契約を破ったからであり、彼がイスラエルの中で恥辱となることをしたからである。』」

●くじのプロセス

神が告げた手順どおりに、

部族ごと → ユダ

ユダの氏族 → ゼラフ人

ゼラフ人から男一人ひとり → ザブディ

ザブディの家族から男一人ひとり → アカン

という風に、同心円状に絞り込まれていきます。

ここには二つの強いメッセージがあります。

罪は「隠し通せない」

　人間の側では完全に隠したつもりでも、主は「くじ」を用いてでも、必ず特定される。

　※箴16:33「くじはふところに投げられるが、すべての決定は主から来る。」

共同体全体が「息をのんで見ている」

　部族 → 氏族 → 家族 → 個人...と絞られていく過程を、イスラエル全体が見守っています。

　これは、アカンだけの物語ではなく、「神の民全体が、罪と聖さについて学ぶレッスン」になっています。

**7:19--21　アカンの罪の「告白」と、欲望のパターン**

**Josh. 7:19** ヨシュアはアカンに言った。「わが子よ。イスラエルの神、**主**に栄光を帰し、主に告白しなさい。おまえが何をしたのか、私に告げなさい。私に隠してはいけない。」

**Josh. 7:20** アカンはヨシュアに答えた。「確かに、私はイスラエルの神、**主**に対して罪を犯しました。私は次のようなことをしました。

**Josh. 7:21** 私は分捕り物の中に、シンアルの美しい外套一着と、銀二百シェケルと、重さ五十シェケルの金の延べ棒一本があるのを見て欲しくなり、それらを取りました。それらは今、私の天幕の中の地面の下に隠してあり、銀もそこにあります。」

●ヨシュアの呼びかけ

「わが子よ」（わが息子よ）と呼びかけている点が印象的です。

・単なる敵としてではなく、「イスラエルの中の一人の息子」として扱い、

・「主に栄光を帰し」「主に告白しなさい」と、悔い改めと真実の告白を促しています。

「主に栄光を帰す」と「告白する」が結びついているのも重要です。

→ 神の前で罪を認め、隠さず明らかにすること自体が、「神の義と真実を認める行為＝神の栄光を認めること」とされているのです。

●アカンの告白の中身（7:20--21）

「見て → 欲しくなり → 取った → 隠した」

という流れが非常に典型的です。

「見た」

　・分捕り物の中に「シンアルの美しい外套」「銀二百シェケル」「金の延べ棒」を「見た」。

　・「シンアル」はバビロン地方の古い名称で、「異国の豪華な服」を連想させます。

　　→ 単なる衣ではなく、「目を引く富・ステータスの象徴」としての外套です。

「欲しくなった」

　・欲望が心の中で膨らむステップ。

　・創世記3章の「女がその木の実を見て...好ましく...賢くしてくれると思った」流れや、ダビデがバト・シェバを見て欲望を抱く流れと重なります。

「取った」

　・行為に踏み出した瞬間。

　・ここで初めて、内面の欲望が「具体的な罪の行為」となります。

「隠した」

　・天幕の地面の下に埋め、さらに銀はその下に。

　・罪はいつも、「隠す」ことを伴う。

　・しかし、天幕の下には隠せても、「主の目」からは隠せない。

**この「見た→欲した→取った→隠した」という流れは、罪が生まれる典型的な心理パターンとして、非常に象徴的です。**

●金額・量のイメージ

・銀200シェケル＋金50シェケル相当

　（1シェケルを約11〜12gとすると、銀2kg強、金は約600g前後のイメージ）

・当時の普通の家庭から見れば、かなり大きな財産です。

→ アカンは、「生涯レベルでの大金」＋「一目置かれる豪華な衣服」を、一気に手に入れようとしたと言えます。

**7:22--23　罪の「現場検証」と、主の前への提示**

**Josh. 7:22** そこでヨシュアは使いたちを送った。彼らは天幕に走って行った。すると、それらはアカンの天幕に隠されていて、銀もその下にあった。

**Josh. 7:23** 使いたちはそれらを天幕の中から取り出し、ヨシュアとすべてのイスラエルの子らのところに持って来て、**主**の前に置いた。

●「走って行った」

・ヨシュアの側は、アカンの告白をすぐに検証します。

・感情的に断罪するのではなく、「事実」を確かめ、そのうえで対処する姿勢も見えます。

●「主の前に置いた」

・単なる「証拠物件」としてではなく、「主の前」に置かれます。

→ この罪は、まず「神に対する罪」であり、その神の前に「何が行われたか」が明らかにされる必要があります。

**7:24--25　アカンへの裁きと、共同体としての関与**

**Josh. 7:24** ヨシュアは全イスラエルとともに、ゼラフの子アカンと銀、外套、金の延べ棒、および彼の息子、娘、牛、ろば、羊、天幕、それに彼のすべての所有物を取って、アコルの谷へ運んだ。

**Josh. 7:25** ヨシュアは言った。「なぜ、おまえは私たちにわざわいをもたらしたのか。**主**は今日、おまえにわざわいをもたらされる。」全イスラエルは彼を石で打ち殺し、彼の所有物を火で焼き、それらに石を投げつけた。

この箇所は、読者にも非常に重く感じられる場面です。

●裁きの範囲

「彼と彼のすべての所有物」が対象。

息子・娘・家畜・天幕・全財産まで含められる。

二つの視点から整理できます。

「ヘレム（聖絶）」の徹底

　「聖絶の物」がイスラエルの陣営に持ち込まれた結果、

　アカンとその所有物全体が「聖絶の対象」とされます。

　→ 神に属するべきものを盗んで自分のものとしたために、「自分と自分の所有物すべて」が神によって「滅ぼし尽くされるべきもの」と宣言される逆転。

家族・所有物の巻き込み

　現代的感覚では、「なぜ子どもまで？」という強い違和感が生じます。

　聖書本文は、「子どもたちが罪に加担していたかどうか」を明示しません。

　・一つの読み方は、「家族全体が天幕の中の隠匿行為を知っていた可能性が高い」という点を重く見る読み。

　・もう一つは、「契約共同体の中で家族全体が一つの単位として扱われる」という古代の家族観・法観を、歴史的背景として受け止める読みです。

いずれにしても、ここで強調されているのは、

神が「半端な処理」をされないこと

罪を共同体から抜き取るために、「徹底的除去」が必要だということ

です。

●「全イスラエル」が石で打ち、焼き、石を投げつけた

・裁きは一部の役人だけでなく、「全イスラエル」が関わります。

→ 罪と裁きの現場は、「共同体全体が見て学ぶ場」でもある。

　これによって、

　「聖絶の物に手を出すことがどれほど恐ろしいか」が、全員の骨身にしみる形で刻み込まれます。

**7:26　石の山と「アコルの谷」― わざわいと、後の希望への伏線**

**Josh. 7:26** 人々はアカンの上に石くれの大きな山を積み上げた。今日もそのままである。**主**は燃える怒りを収められた。それで、その場所の名はアコルの谷と呼ばれた。今日もそうである。

●「石くれの大きな山」

・エリコ（6:26）でも石塚・記念が出てきますが、ここでも「記念碑」が築かれます。

・役割は逆で、

　・エリコの石塚＝主の勝利・約束の成就の記念

　・アコルの石塚＝罪と裁き・わざわいの記念

●「今日もそのままである」

・この時点で書かれている「今日」まで、眺望可能な記念として残っているということ。

→ イスラエルがその地を通るたびに、

　「アカンの罪」「神の聖さ」「隠れた罪の恐ろしさ」を思い起こす教育的役割を果たしたと言えます。

●「アコルの谷」

「アコル」（עָכוֹר）は「わざわい・混乱・災い」を意味する語根と関係があり、

ヨシュアの「なぜ、おまえは私たちにわざわい（アカル）をもたらしたのか」との掛け合わせ（言葉遊び）になっています（「アカン」とも音が近い）。

興味深いのは、後の預言でこの「アコルの谷」が別の意味を帯びることです。

ホセア2:15では、「アコルの谷」が「望みの門」と言い換えられます。

　→ かつての「わざわいの場所」が、「回復と希望の出発点」に変えられる、という預言的逆転。

ヨシュア記の段階では、ここは「罪と裁きとわざわいの谷」です。

しかし、後の預言者は同じ地名を用いて、

「神は、わざわいの場所から希望を開くことができる」と語るようになります。

→ 神学的には、「罪と裁きの記念」が、後に「恵み・回復の入り口」の象徴に反転させられるという、大きな流れの起点になっているとも読めます。

**全体のまとまり：アカン事件後半が教えること**

**罪は隠し通せない**

くじを通してでも神は特定される。

「見て→欲して→取って→隠した」罪は、「暴かれ→告白され→徹底的に処理される」流れに必ず向かう。

**告白は神への「栄光」**

ヨシュアは「主に栄光を帰し、主に告白しなさい」と言う。

罪の告白は、自分を潰す行為ではなく、「神の義と真実を認める行為」として位置づけられる。

**聖絶の徹底と共同体の聖さ**

アカンとその所有物が徹底的に処理されるのは、

残酷さというより、「神の民のただ中から、ヘレム（聖絶の物）を根こそぎ除去するため」。

聖さの共同体には、「罪を甘く見て放置する」という選択肢は与えられていません。

**記念は「恵み」だけでなく「警告」も刻む**

アコルの谷の石の山は、「こんなことが二度と起こらないように」という厳しい記念。

しかし、後の預言においては、同じ名前が「望みの門」へと転用される。

神は、わざわいの場所さえ、恵みの物語に組み込みうる方である、という長いスパンの視点につながります。

この7:16--26は、アカン個人の悲劇であると同時に、

「神の民が罪をどう扱うべきか」「聖さと裁き・そして後の希望」を教える、非常に濃い部分として読むことができると思います。
