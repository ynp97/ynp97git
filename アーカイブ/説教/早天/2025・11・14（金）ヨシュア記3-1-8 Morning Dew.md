2025・11・14（金）ヨシュア記3:1-8 Morning Dew

全体像だけ先に整理してから、節ごとに見ていきます。

この3:1--8は、大きく二つに分かれます。

1--6節：民と祭司への「準備の命令」

7--8節：ヨシュアに対する主の「約束と具体命令」

ヨルダン渡河というクライマックスの直前、「どう備えるか」と「誰に従うか」が丁寧に整えられている場面です。

3:1　シティムからヨルダン川へ――決断と移動の「朝」

**Josh. 3:1** ヨシュアは翌朝早く起き、すべてのイスラエルの子らとともにシティムを旅立ち、ヨルダン川のところまで来て、それを渡る前にそこに泊まった。

ヨシュアは翌朝早く起き、...シティムを旅立ち、ヨルダン川のところまで来て、それを渡る前にそこに泊まった。

●「翌朝早く起き」

ヘブライ語では「שָׁכַם（シャーカム）」が使われます。アブラハム、モーセなども、重大な従順の行動に出るときに「朝早く起き」ます（創22:3, 出8:20など）。

→ 意味合い：ぐずぐずせず、信仰をもって即座に従う姿勢。

●「シティム」

ヨルダン東側・モアブの平地の宿営地。民数記25章での不信仰・淫行・偶像礼拝の舞台でもあります。

→ そこから「約束の地の入口（ヨルダン川）」へ移動するのは、霊的にも「古い不忠実の地」から「約束の成就」へ進む象徴的な一歩。

●「それを渡る前にそこに泊まった」

すぐ渡らず、川岸で一泊。

・軍事的には、渡河前の最終整備・偵察・隊列の確認の時間。

・霊的には、「約束の地を前に一度立ち止まり、神の御業を待つ」ための間。

→ 信仰の歩みでも、決断した直後にすぐ「成就」ではなく、「待つ一夜」「心を整える時間」がしばしば置かれます。

**決心は早いが、主の前で鎮まる時がある。**

3:2　「三日後」――備えのための猶予

**Josh. 3:2** 三日後、つかさたちは宿営の中を巡り、

三日後、つかさたちは宿営の中を巡り、

●「三日後」

三日という期間は、旧約で「準備・試練・転換点」によく登場します（出19:11のシナイ準備、ヨナ3日間など）。ここでは

・川辺に着いてから三日＝隊列の整理・心の備え・恐れと期待が高まる時間。

●「つかさたちが宿営の中を巡り」

指導者たちが各テントを回り、具体的な指示を徹底させる。

→ 神の大きな御業の前に、「全員が同じ指示を聞き、同じ行動パターンに揃えられる」ことが重視されている。

3:3　契約の箱に従え

**Josh. 3:3** 民に命じた。「あなたがたの神、**主**の契約の箱を見、さらにレビ人の祭司たちがそれを担いでいるのを見たら、自分のいる場所を出発して、その後を進みなさい。

「...契約の箱を見、さらにレビ人の祭司たちがそれを担いでいるのを見たら、自分のいる場所を出発して、その後を進みなさい。」

●「あなたがたの神、主の契約の箱」

ヘブライ語で「אֲרוֹן בְּרִית יְהוָה（アロン・ベリット・ヤハウェ）」

・「契約」＝シナイで結ばれた主とイスラエルの関係。

・箱そのものが「主の臨在の象徴」であり、「主ご自身が先頭に立って行かれる」ことを具体的に見せる道具。

●「レビ人の祭司たちがそれを担いでいる」

通常、契約の箱を担ぐのはケハテ族ですが、ここでは「祭司たち」と強調される。

・約束の地への最初の一歩は、特別な聖なる儀礼として扱われ、最も聖別された者たちが担い手となる。

・民は自分たちの知恵や軍事力ではなく、「祭司に導かれる神の箱」の後を進む。

●「その後を進みなさい」

ここで示される進軍の基本原則：

まず神の臨在（契約の箱）が動く

祭司がそれを担って従う

民はそれに従って動く

→ 「神が先、私たちは後」。信仰の歩みの基本パターンが視覚的に示されている。

3:4　2000キュビトの距離と「未知の道」

**Josh. 3:4** あなたがたが行くべき道を知るためである。あなたがたは今まで、この道を通ったことがないからだ。ただし、あなたがたと箱の間に二千キュビトほどの距離をおけ。箱に近づいてはならない。」

「...あなたがたが行くべき道を知るためである。あなたがたは今まで、この道を通ったことがないからだ。ただし、あなたがたと箱の間に二千キュビトほどの距離をおけ。箱に近づいてはならない。」

●「行くべき道を知るため」

物理的な意味：

・数十万〜百万人規模の民が、広い視界で「箱の位置」を見失わないように、先頭と民の間を充分に空ける。

・箱が前方に一定距離にあることで、全体の隊列が「どちらへ向かうのか」をはっきり認識できる。

霊的な意味：

・「あなたがたは今まで、この道を通ったことがない」

　ヘブライ語では「キ ロ アバルトֶム バデレフ ミトモル シルショム」に近い表現で、「きのうも一昨日も通ったことのない道」という慣用句。

→ 完全に未知の領域。経験も実績も頼りにならず、「主の導き」だけが頼り。

●「二千キュビトほどの距離」

1キュビト ≒ 45cm と仮定すると、2000キュビト ≒ 約900m。

意味合い：

・視界に十分入る距離

・しかし、決して「近づきすぎない」距離

神学的には

【聖なる距離】

　契約の箱は神の聖さを象徴し、気安く近づいてはならない。サムエル記で箱に軽率に触れたウザが打たれる場面（2サム6章）とも響き合う。

【方向感覚のための距離】

　近すぎると先頭の一部しか箱を見えないが、距離を空けることで、全体が方向を見失わない。

→ 神との関係は、「親しく近づいてよい」側面と、「畏れ敬う距離」が必要な側面の両方を持っていることの象徴。

3:5　「自らを聖別しなさい」――ヨルダン渡河前夜の心構え

**Josh. 3:5** ヨシュアは民に言った。「あなたがたは自らを聖別しなさい。明日、**主**があなたがたのただ中で不思議を行われるから。」

「あなたがたは自らを聖別しなさい。明日、主があなたがたのただ中で不思議を行われるから。」

●「自らを聖別しなさい」

ヘブライ語：הִתְקַדְּשׁוּ（ヒトカッデシュ―）

・自動詞・再帰的な形「自分自身を聖なるものとして整える」。

・出エジプト記19章でシナイ山への備えをするときと同じ表現。

→ 新たな段階（シナイ＝契約の確立、ヨルダン渡河＝約束の地の実際の取得）に入る前には、必ず「聖別」が要求される。

具体的には

・身を洗う、衣服を清める

・偶像・不浄なものから距離を置く

・心を悔い改めに向ける

などの儀礼＋内面的整え。

●「明日、主が...不思議を行われる」

「不思議」＝נִפְלָאוֹת（ニフレアオート）系の語。

・人間の力や自然法則を越えた、神の超自然的な御業。

・ここでは、ヨルダンの水がせき止められる奇跡が具体的な「不思議」。

ポイント

・神の奇跡は、「ただ勝手に起こる」のではなく、民の側の「聖別」とセットで語られている。

　→ 「まず悔い改めと聖別」「そのうえで神の御業を待つ」という流れ。

3:6　祭司に対する命令と即時の従順

**Josh. 3:6** ヨシュアは祭司たちに「契約の箱を担ぎ、民の先頭に立って渡りなさい」と命じた。そこで彼らは契約の箱を担ぎ、民の先頭に立って進んだ。

「契約の箱を担ぎ、民の先頭に立って渡りなさい」...彼らは契約の箱を担ぎ、民の先頭に立って進んだ。

●ヨシュアの命令の内容

・「先頭に立って」＝最も危険なところに出るのが祭司と箱。

・前線に出るのは将軍や兵士でなく「神の箱」と「祭司」。

●「渡りなさい」

まだヨルダンは分かれていない段階（※水が止まるのはこの後の節）。

→ 祭司たちは「水はまだ流れているのに、『渡れ』と言われたから足を踏み入れる」＝御言葉に先行して従う信仰が要求されている。

●「彼らは...進んだ」

余計な議論も条件交渉もなく、即座に従っている描写。

→ ここで描かれる祭司たちの従順は、後に民に示される「信仰の手本」となる。

**これができるようになっていたことが重要である。**

3:7　主によるヨシュアの公的承認

**Josh. 3:7 主**はヨシュアに告げられた。「今日から全イスラエルの目の前で、わたしはあなたを大いなる者とする。わたしがモーセとともにいたように、あなたとともにいることを彼らが知るためである。

「今日から全イスラエルの目の前で、わたしはあなたを大いなる者とする。わたしがモーセとともにいたように、あなたとともにいることを彼らが知るためである。」

●「今日から...あなたを大いなる者とする」

ヘブライ語で「始める（אָחֵל）」＋「大いなる者にする（גִדֵּל）」という組み合わせ。

・ヨシュアの偉大さは、彼自身のカリスマや能力ではなく、「主が彼を大きくする」という形で与えられる。

・この日（ヨルダン渡河）は、ヨシュアの指導者としての「公式な権威確立の日」。

●「わたしがモーセとともにいたように」

・ヨシュアはモーセの後継者として立てられているが、民にとってモーセは絶対的な存在。

・神は、「同じレベルでヨシュアを支える」と宣言することで、民の不安を取り除き、ヨシュアへの信頼を確立させる。

→ ヨルダン渡河は、モーセ時代の「紅海渡河」の再現のような位置づけで、

　「指導者交代」＋「同じ神の同じ力が働いている」というメッセージを見える形にするイベント。

3:8　ヨルダンの中に立ち続けよ――信仰の具体的な一歩

**Josh. 3:8** あなたは契約の箱を担ぐ祭司たちに『ヨルダン川の水際に来たら、ヨルダン川の中に立ち続けよ』と命じよ。」

「ヨルダン川の水際に来たら、ヨルダン川の中に立ち続けよ」

●具体命令

・祭司たちは「水際で止まる」のではなく、「川の中に入って立ち続ける」。

・後の3:15--16で明らかになりますが、ヨルダンは「刈り入れの頃で、岸一杯にあふれ」ている時期。

→ 人間的には「一番危ない時期」に、「一番危ない場所」に立ち続けよという命令。

●「立ち続けよ」

・「とりあえず足を入れて様子を見る」ではなく、「そこにとどまり続ける」。

・信仰は「一瞬の感情的ジャンプ」で終わるものではなく、「神の御言葉に立ち続ける持続的な態度」として描かれている。

●神学的ポイント

祭司・契約の箱が「水の中に立つ」ことで、民は「乾いた川底」を通過できるようになる（この後の展開）。

　→ 神の臨在が、死と危険の象徴である水をせき止め、道を作る。

主の奇跡は、「人間の側が危険の中に足を踏み入れる信仰の行為」と結びついている。

　→ 「水が止まったら渡る」ではなく、「渡り始めるときに水が止まる」。

**まとめ：3:1--8から見える信仰のパターン（説教・適用のヒント）**

**未知の道への出発（3:1--4）**

・「あなたがたは今まで、この道を通ったことがない」

→ 人生の新しい段階・新しい使命・新しい季節に入るとき、私たちの経験は役に立たない部分が多い。

→ 「神の臨在（箱）」に目を注ぎ、距離を保ちつつも、その後を進む必要がある。

**聖別と奇跡（3:5）**

・「自らを聖別しなさい」＋「明日、主が...不思議を行われる」

→ 神の御業を求める前に、自分の側で「整えるべきもの」がある。

→ 悔い改め・優先順位の見直し・偶像からの決別なしに、「奇跡だけ」を求めるのは順序が逆。

**リーダーと民の信頼関係（3:6--7）**

・主がヨシュアを「大いなる者とする」

→ 健全な霊的リーダーシップは、「神が立て、神が承認した」ことが民に見える形で示される必要がある。

→ リーダー本人も、自分の権威を「自分の手柄」ではなく、「神の共におられるしるし」に根ざして理解することが大切。

**川の中に立ち続ける信仰（3:8）**

・水が引いてからではなく、「まだ流れているときに足を踏み入れる」。

・しかも、「そこに立ち続けよ」と命じられている。

→ 私たちが与えられている約束に従って、危険と思える場所でも「引かずに立ち続ける」ことを通して、神は道を開かれる。

この3:1--8は、単なる「ヨルダン渡河前の準備」ではなく、

「未知の道へ踏み出すときの信仰のパターン」を、

視覚的・象徴的に示している導入部だと整理できます。

未知の道を進む時、神を信じ決心し、整え、神の後についてゆく。

まずは、神に従うところから始めなければならない。

奇跡は私たちの思うメリットのためではなく、神を信じるという恵みのためにあるから、

神を先にし、勇気を持って歩むという順序の中に生まれてくる。
