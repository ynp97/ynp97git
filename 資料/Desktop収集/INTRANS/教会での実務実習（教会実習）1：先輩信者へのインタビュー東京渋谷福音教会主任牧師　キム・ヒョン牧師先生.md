# 教会での実務実習（教会実習）1：先輩信者へのインタビュー東京渋谷福音教会主任牧師　キム・ヒョン牧師先生

	1.	どのようにして、イエスとの深い関係を維持していますか。
	2.	最も好きな聖句は何ですか。また、なぜその聖句は大切ですか。
	3.	もし、「新しくクリスチャン」になった——回心した——当時に戻り、当時の自分に忠告をすることができるなら、どのような忠告をしますか。また、なぜですか。
	4.	クリスチャンになってからの時間を振り返ると、神様との関係はどのように成長してきたと思いますか。
	5.	今までの人生で経験したことの中で、最も困難な瞬間とはどんな時でしたか。その時、キリストの助けを、どのような形で経験しましたか。

1.まず1日の始めをQTで始める。イエス様との深い関係を維持するのは、他でもなく「御言葉」と「祈り」である。

2.詩編37:5-6、大学生の時から神学大学院まで暗唱していた。特に「あなたの道を主に委ねよ」という部分からそうだ主に道を委ねようと思ったから。牧会者となり、最も好きな聖句はローマ8:35-39その中でも8:37となった。「圧倒的な勝利者」として下さる。ただ神様からの一方的な恵みで、またその愛で引き離す事は無いと言って下さるのは、メッセンジャー として働き人として何の資格も無い自分にとって大きな励ましを感じるから。

3.幼い頃から教会に熱心に通っていたが、イエス様と人格的に出会い「イエス様が神様の息子であり、私の救い主であり、人生の主です。」と告白出来たのは、大学一年生の時、CCCで「四つの法則」を先輩から聴いた時。その時の自分に忠告するとしたら、一生懸命に訓練や学びをしていたが、形式的な所があったので、「もっと主への真心をもって訓練や学びを行いなさい。」また、将来の進路の為にCCCを辞めるか迷ったが、その時迷わずにいたら働き人になる時間を早められたと思うので、それを伝えたい。ただ、その当時の働きなどにおいて後悔は全くない。

4.キリスト教をただ宗教としか考えていなかった自分が、確固たる思いで今神様は生きておられ、今日死んでも神様の身許にいけると告白出来るようになった。それを否定する事が出来なくなった自分自身を省みる時、日本に宣教師として遣わされて、苦労はあったが、任せられる働きも段々と大きくなり、その各段階を通して日々神様との関係が成長したんだなと感じる。そのように成長出来たのは、自分が何者でもないのに、何か優れている訳でもないのにただただ神様の恵みによるという感謝の告白しか出て来ない。

5.日本に宣教師として初めて来た時、開拓教会でミニストリーをしていた時、色々な困難があり、どうしても受け入れ難い事情があり、その教会を去らざるを得なかった。その後韓国に戻らないといけないのかと悩み、最も苦しかった。次の進路について神様に導きを求めていた時、祈祷院での集会で賛美リードを頼まれた。実は行きたくなく、しかも100人来ると聞いていたが、実際は10数名しかいなかった。しかし、1,000人がいるかのように真心をもって主に賛美し、礼拝をした。そうするとそこに来ていた牧師先生がその姿を見て心打たれ、「副牧師として招きたいが、今は来てもらう余地がないので、いつか必ず招聘するからそれまで持ち堪えて」と言われた。そんな事言われてもと思っていたが、それが突然1ヶ月後に副牧師が必要になったと連絡があり、招聘された。その先生もその集会に行きたくなかったそうで、後に「あなたに会う為に集会に導かれたのだ」と言って下さり、逆転の神様はこのように働かれるんだと実感し、真の礼拝者を求め探しておられる神様に応答した事によって、次のミニストリーの場へと神様が導いて下さったんだとキリストの助けを経験した。
