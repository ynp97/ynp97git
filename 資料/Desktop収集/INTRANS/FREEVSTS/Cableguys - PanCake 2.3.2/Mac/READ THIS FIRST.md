# READ THIS FIRST


=====================================================
macOS
=====================================================

IMPORTANT: For Ableton Live and Cockos REAPER, please use the VST version, as Live and REAPER had minor issues with automation with Audio Units plugins in our tests.


PanCake 2 is an audio plugin in VST 2, VST 3, AU and AAX formats. You can install and use it alongside PanCake 1.

To install PanCake, simply double-click the .pkg package file (looks like an open box) and follow the instructions.

If you don't see PanCake in your DAW's plugin list, please restart your computer. Some DAWs (eg, Logic Pro X and Ableton Live) may not recognize the plugin otherwise.
