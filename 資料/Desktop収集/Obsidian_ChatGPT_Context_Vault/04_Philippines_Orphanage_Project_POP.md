# 04_Philippines_Orphanage_Project_POP

# Philippines Orphanage Project / POP

## 名称
- 日本語：フィリピン孤児院プロジェクト
- 英語：Philippine Orphanage Project
- 略称：POP

## 目的
- フィリピンに孤児院を設立する。
- 孤児に信仰・教育・生活の場・仕事への道筋を与える。
- 成人後も帰れる「実家」のような場所を目指す。
- 日本の若者、特にドロップアウトした若者が海外で就労・生活体験できる場所にもする。

## 基本構想
- 資金ゼロに近い状態から始める。
- 宿泊・カフェ・ドミトリー等で収益化し、孤児院運営につなげる。
- ソーシャルファーム型。
- 教会外活動が増えるため、教会員に誤解されない説明が必要。
- これはフリースクールの延長であり、信仰的な神の導きへの応答と考えている。

- DGTからシキホール。

### Leyte新計画
- 場所：Libas, Merida, Leyte
- Ptr. Denis土地優先貸与。
- 建設：一階建て。二階建て案は撤回。
- 8月着手可能性。
- 手続き：
  - Draftsman
  - 建築士署名
  - 許可
  - 土盛り
- 資金：2か月以内送金予定という話があった。
- 交通：滞在地からタクロバン空港まで約3時間。
- 活動：子ども食事・プール等。

## 神学的・宣教的意識
- 「カナンに入れなかったイスラエルであってはならない」という思い。
- 説教で語ってきたことを牧師自身が実践するという意識。
- 忙しすぎる現実と、アメリカや海外との関係構築の必要性の間で葛藤がある。
