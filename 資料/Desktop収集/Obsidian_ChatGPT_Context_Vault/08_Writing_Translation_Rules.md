# 08_Writing_Translation_Rules

# 文章作成・翻訳ルール

## 日本語整形
- 音声入力原稿を、語順を大きく変えずに整える依頼が多い。
- 句読点、誤字、重複、言い直しを自然に整理する。
- Wordに直接貼って違和感のない形を希望。
- 前置き不要。

## 説教・聖書原稿
- 本文忠実。
- 対象に合わせて、高校生向け・大学生向けに調整。
- 高校生向けは簡潔に。
- 「って」「よね」は避ける。
- デスマス調。

## 英訳
- 日本語構造に忠実。
- 要約しない。
- 挨拶や余計な説明を加えない。
- 福音派の用語感を保つ。
- 聖書引用はESV。
- 本文のみの出力を求めることが多い。

## 事務文面
- 断り文、依頼文、保護者宛、学校連絡文を作成。
- 丁寧だが重すぎない文体を好む。
- 「助言」は上から目線に感じる場合があるため注意。
