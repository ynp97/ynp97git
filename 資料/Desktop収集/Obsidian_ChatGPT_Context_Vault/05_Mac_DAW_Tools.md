# 05_Mac_DAW_Tools

# Mac・DAW・ツール環境

## 現在のMac
- MacBook Pro 14インチ
- M5
- メモリ32GB
- SSD 2TB
- Nano-texture
- 日本語JIS
- 使用開始後、「買って良かった」と評価。
- 15インチに戻ると、14インチの良さを感じる。
- Nano-textureは完全に映り込みが消えるわけではないが、良さはある。
- 15インチはもっさりして見える。

## Mac運用で詰まったこと
- iCloudとローカルフォルダの概念。
- デスクトップやダウンロードの場所。
- Finderサイドバー整理。
- スクリーンショット保存場所変更。
- 移行アシスタントが終わらない問題。
- 普通のUSB-CケーブルとThunderboltの違い。
- iCloud同期を切ると表示が消えることへの理解。

## Cubase
- ウインドウレイアウトをショートカットに記憶。
- プラグイン整理・非表示。
- VST Plug-in Information が見つからない問題。
- AirPods出力、MacBook内蔵マイク入力の組み合わせ。
- 集約デバイスの意味。
- 普段のMac音声設定とDAW用設定を分けたい。
- MIDIデータをLogicと受け渡し。
- Logicで.midからコード検出。
- CubaseとLogicでドラムマップ共有可否。

## Logic Pro
- テンポ変更時にコードトラックや選択範囲が追従しない問題。
- 最終的に「磁石」関連の設定で解決。
- 表示が複雑すぎる場合、シンプル表示の可能性も検討。

## OBS
- マイク設定が外れる。
- 音声入力されない時がある。
- どこで固定するかを確認。

## 音楽・映像
- Cubase、Logic、Final Cutに関心。
- AirPodsでモニターし、MacBook内蔵マイクでメモ的に録音したい。
- 音質は問わない用途。
- VCV Rackを初心者向けに毎日10分程度で学ぶ案。
- ギター練習：1日30分リフ、スケール10分、TAB希望。
