# 00_INDEX

# ChatGPT セッション引き継ぎ Vault

これは、ChatGPT側で参照できる範囲の過去セッション内容を、Obsidianで扱いやすいように整理したものです。

## 重要な制限
- 全セッションの逐語ログではありません。
- ChatGPTが現在参照できる「会話要約」「長期記憶」「最近の履歴」「一部ファイル情報」をもとにした整理版です。
- 正確な原文が必要な場合は、ChatGPTのデータエクスポートから会話履歴を取得して、別途Obsidianに取り込む必要があります。
