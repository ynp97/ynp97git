# README_最初に読んでください

ポケカ戦績 SwiftUI ネイティブ版 v0.1

これはブラウザを使わない、macOS用のSwiftUI試作版です。
HTMLラッパーではありません。
データはSQLiteに保存します。

保存先:
~/Library/Application Support/PokecaRecordsSwiftUI/records.sqlite3

開き方:
1. MacにXcodeをインストールしてください。
2. このフォルダの Package.swift をダブルクリック、またはXcodeで開いてください。
3. 画面上部の実行先を「My Mac」にします。
4. Run ▶︎ を押します。

現在できること:
- 戦績入力
- 日付・時刻記録
- 勝敗、先攻後攻、初手のボタン選択
- デッキ選択
- 戦績一覧
- 編集、複製、削除
- 相性表
- 実勝率
- 補正勝率
- 信頼度表示
- 最低試合数フィルター
- デッキ画像登録
- JSONバックアップ読み込み・書き出し
- CSV書き出し

注意:
これはApp Store提出前の試作プロジェクトです。
App Storeに出すには、正式なXcode Appプロジェクト化、アイコン、署名、サンドボックス、プライバシー説明、テストが必要です。

HTML版から移行する場合:
HTML版でJSONバックアップを書き出し、このアプリの「バックアップ」タブからJSONを読み込んでください。
