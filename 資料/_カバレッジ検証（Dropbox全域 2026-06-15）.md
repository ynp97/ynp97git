# 🔎 取り込みカバレッジ検証 ── Dropbox 全域（2026-06-15）

> [!info] この検証の意味
> 「散らばった一次資料をすべてVaultの参照可能なMarkdownに鏡像化する」というゴールに対し、**鏡像に抜けがないか**を数字で裏取りした検品結果。今回はユーザー指示により **Dropbox 全域** を対象にした（従来の取り込みスコープ＝個人フォルダ＋Desktop より広い）。
> 関連: [[_作業ログ・引き継ぎ]] ｜ [[_索引（何がどこにあるか）]]

## 方法と前提（重要）

ファイル名の一致（拡張子を除いた**名前ベース**）で突合した。中身の照合ではない。

- 対象文書＝docx / doc / pdf / xlsx / xls / pptx / ppt / txt / rtf / odt / csv / dotx
- 除外＝画像(jpg/png/psd/ai/eps等)・音声/動画(wav/mp3/mp4等)・実行/ビルド成果物(exe/app/nib/strings/framework等)・ショートカット(lnk)
- 「競合コピー（LAPTOP-… の競合コピー）」サフィックスや空白・ダッシュ表記ゆれは正規化して比較
- **名前一致のため、別名で取り込まれた文書は「未取り込み」に誤判定されうる**（過大に出る方向）。逆に同名の別物は「取り込み済み」と誤判定されうる。数字は目安。

## 既存スコープ内の取りこぼし（要確認）

「個人フォルダ＋Desktop」は取り込み済みのはずだが、個人フォルダで **26点** が名前未一致だった。主なもの:

- **歌詞 docx/rtf/odt**（`!!LYRICS/feit/` のオートサクリファイス・夢現・最後の最後・ミルクと蜂蜜 ほか、ライブ歌詞.pptx 等） … 曲名は[[🎵 歌詞インデックス]]やジャーナルに登場するが、**個別の歌詞原本ファイルはmd化されていない**。創作（四つの柱）の一次資料なので取り込み推奨。
- TODO/メモ txt（`ゆうちょ口座番号.txt`・`FAVplaylist…txt`・`darkstars live info.txt`）
- PDF（`放送大学bangumihyo`・航空券控え `BP_…NRT-CEB…`・`NPO202106_manual`・DayOneエクスポート・各種マニュアル） … マニュアル類・予約控えは**除外候補**。

→ 全件は `資料/_未取り込み一覧（Dropbox全域 2026-06-15）.tsv` に保存（1,627行・パスと拡張子）。

## 解釈と次の判断

1. 従来スコープ（**個人フォルダ・説教・Desktop**）は概ね鏡像化済み。「全域」で大量の未取り込みが出たのは、**KGC共有・学研・教会共有・神学校・黙示録・確定申告**など、これまで意図的にスコープ外だった institutional フォルダを今回初めて含めたため。
2. よって次の判断は「**これらをVaultに取り込むべきか**」。性格別に分けると:
   - 取り込み価値が高い … 神学校・黙示録・説教の取りこぼし（信仰の柱）、歌詞原本（創作の柱）
   - 業務アーカイブ … KGC共有・学研・KG・教会共有（量が多い。必要分のみ選別が現実的）
   - 除外でよい … 確定申告（機微・実務）、各種マニュアルpdf・予約控え
3. pdf 1,115点は、Desktop分（191点）と合わせて `pdftotext` 一括変換すれば名前一致は大きく改善する見込み。

> [!question] 確認したいこと
> 「全域を漏れなく鏡像化」がゴールか、それとも「四つの柱（信仰・創作・教育・POP）に関わる一次資料のみ」でよいか。後者なら確定申告・マニュアル類は恒久的に除外として扱う。
